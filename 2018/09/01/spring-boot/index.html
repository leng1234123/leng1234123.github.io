SpringBoot之旅

2018-09-01   Java Spring SpringBoot 

一、什么是SpringBoot

SpringBoot是由Pivotal团队提供的全新框架，其设计目的是简化Spring应用的初始搭建配置以及开发过程；使用Spring Boot我们可以轻松创建独立的，快速构建生产级的基于Spring的应用程序。

二、为什么要使用SpringBoot

  1. 可以创建独立的Spring应用程序,进行快速构建开发
  2. 直接嵌入Tomcat，Jetty或Undertow（无需部署WAR文件）
  3. 不需要编写XML配置文件，尽可能自动配置Spring和第三方库等

三、SpringBoot快速入门

本章环境要求：jdk1.8 + IntelliJ IDEA工具 + Maven环境

创建项目 File->New->Module


然后选择Spring Initializr，之后一直Next，然后Finish完成创建


我们可以看到项目目录


pom.xml文件可以添加自己需要的Jar包或者更改SpringBoot版本号。

然后打开DemoApplication启动类，我们可以看到@SpringBootApplication，它是SpringBoot最重要的一个注解，用于快捷配置启动类。
我们在添加一个@RestController注解定义Controller类，相当于@Controller + @ResponseBody ，主要是为了使 http 请求返回数据格式为 json 格式。
@GetMapping注解来配置 url 映射get请求方式，可以作用于类上，也可以在方法上。

选中main方法右击Run

可以看到控制台输出(Started DemoApplication in 3.043 seconds)，说明启动成功！
